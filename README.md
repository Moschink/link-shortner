

# 🔗 Shortly – Link Shortener

A responsive web app for turning long URLs into clean, shareable links with a modern UI and smooth user experience.

---

## 🚀 Features

* Shorten links instantly
* One-click copy
* Fully responsive design
* Clean, user-friendly interface

---

## 🛠️ Tech Stack

* HTML5
* CSS3 + Bootstrap 5
* JavaScript

---

## 📦 Usage

```bash
git clone https://github.com/your-username/shortly.git
cd shortly
```

Open `index.html` in your browser.
